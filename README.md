### 使用

```sh

git clone https://github.com/haokur/electron-vue3-chatgpt
cd electron-vue3-chatgpt
npm install

# 本地启动
npm start

# 打包桌面端
npm run build
```

### 特性

- 极简，所有操作通过状态栏图标或快捷键操作
- 全局快捷键显示隐藏窗体
